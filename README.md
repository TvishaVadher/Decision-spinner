# Decision Spinner
## An interactive decision-making application built using Python and Tkinter, focused on clean UI design and accurate rotation logic.
## Features
1. Pastel-themed user interface
2. Smooth spinning wheel animation
3. Fixed arrow pointer for accurate selection
4. Randomized rotation with correct visual mapping
5. Editable default options
6. Add custom options dynamically
## How It Works
1. The wheel rotates by a random number of degrees on each spin.
2. The arrow remains fixed at the top.
3. The selected option is determined by calculating which slice lies under the arrow after rotation, ensuring visual and logical consistency.
## Technologies Used
1. Python 
2. Tkinter
